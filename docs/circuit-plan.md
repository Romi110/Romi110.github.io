## Beginner — 75-Day Progressive Plan

### Philosophy
One or two exercises at a time. Never add a new movement until the current ones feel automatic.
Every day has a clear type and focus — variety within weeks prevents boredom while still building the pattern.
Core work (Dead Bug) introduced from Day 2 and recurring throughout.
Rest as long as needed between sets — form over pace, always.
Rest days are distributed mid-week intentionally — not always at the end — to support recovery when volume is highest.

### Layout
Every day is its own card. 7 cards per row = 1 week per row.
Week sidebar shows: week number, title, subtitle, coach note.
Day card colors signal training focus:
- Orange = hinge / foundation / conditioning
- Blue = squat
- Amber = pull
- Green = push
- Teal = carry / recovery
- Purple = skill
- Grey = rest

### Exercises Introduced (in order)
1. **KB Deadlift** — Day 1
2. **Goblet Squat** — Day 1

4. **Two-Handed Swing** — Day 3 (practice only, light weight)

## Intermediate — 7-Day Split

### Philosophy
Heavy kettlebell days separated by light cardio or conditioning days.
Muscles get 48+ hours before being hit again.
Each heavy KB day has 3 options (A/B/C) — same muscle target, different style or intensity.

### Weekly Schedule

| Day | Type | Focus |
|---|---|---|
| Monday | Heavy KB | Hinge / Posterior Chain |
| Tuesday | Light Cardio | Active Recovery |
| Wednesday | Heavy KB | Push / Shoulders / Triceps |
| Thursday | Moderate KB | Conditioning — Swings + Carries |
| Friday | Heavy KB | Pull / Back / Biceps |
| Saturday | Heavy KB | Squat / Legs |
| Sunday | Rest | Mobility / Halo / Stretching |

---

### Monday — Hinge / Posterior Chain

**Option A — Swing Focus:** Two-Handed Swing 5×20 · SA Deadlift 4×10 each · Carry 4×45s

**Option B — Clean Focus:** Clean 5×6 each · Single-Leg Deadlift 3×8 each · Swing 3×15

**Option C — Snatch Intro:** Clean 3×5 (warm-up) · Snatch 4×4 each · SA Deadlift 3×10 each

---

### Tuesday — Light Cardio / Active Recovery

Not structured — choose one: easy run/cycle · jump rope · KB flow (Halo → Goblet Squat → Carry) · Yoga/mobility

---

### Wednesday — Push / Shoulders

**Option A:** Overhead Press 5×5 each · Push Press 3×6 each · Carry 3×45s

**Option B:** Tall Kneeling Press 4×12 each · Overhead Press 3×10 each · Swing 3×15 (active rest)

**Option C:** Clean & Push Press 4×5 each · Overhead Press 3×8 each · Thruster 3×8 · 3 rounds, 90s rest

---

### Thursday — Conditioning (Moderate KB)

**Option A:** Swing ladders: 10, 15, 20, 15, 10 reps · Carry 4×40s

**Option B:** Timed: Swing 30s / Goblet Squat 30s / Carry 30s — 15s rest between · 4 rounds · 60s between rounds

**Option C:** Single-Handed Swing 4×12 each · SA Deadlift 3×10 each · Carry 4×40s

---

### Friday — Pull / Back

**Option A:** Single-Arm Row 5×8 each · High Pulls 4×10 · Carry 3×45s

**Option B:** Clean 4×6 each · Single-Arm Row 4×10 each · Suitcase Row 3×8 each

**Option C:** Circuit: Clean 4×6 → High Pulls 3×12 → Row 3×10 each → Carry 3×40s · 3 rounds · 90s rest

---

### Saturday — Squat / Legs

**Option A:** Goblet Squat 5×8 (heavy) · Racked Squat 4×8 each · Carry 3×45s

**Option B:** Racked Reverse Lunge 4×8 each · Goblet Squat 4×10 · Side Lunge 3×8 each

**Option C:** Circuit: Goblet Squat 4×10 → Racked Reverse Lunge 3×8 each → Swing 3×15 → Carry 3×40s · 3 rounds · 90s rest

### Sunday — Rest / Mobility

Hip 90/90 rotations · World's greatest stretch · Halo (light) · Thoracic rotations · Dead hang if available

## Advanced — Full-Body Non-Stop Circuits

### Philosophy
Pick one circuit. Complete all exercises with no rest between movements.
Rest 2–3 minutes between rounds. Do 3–5 rounds = complete workout.
Every circuit hits the entire body.

---

### Circuit 1 — The Chain

Classic movement-pattern flow. Hinge → push → pull → squat → carry. One bell.

| Order | Exercise | Reps |
|---|---|---|
| 1 | Two-Handed Swing | 15 |
| 2 | Clean | 6 each |
| 3 | Overhead Press | 6 each |
| 4 | Single-Arm Row | 8 each |
| 5 | Goblet Squat | 10 |
| 6 | Farmer's Carry | 45s |

Rest 2–3 min. Repeat 3–5 rounds.

---

### Circuit 2 — The Grind

Heavy, slow, strength-focused. Maximum tension every rep.

| Order | Exercise | Reps |
|---|---|---|
| 1 | Single-Arm Deadlift | 8 each |
| 2 | Overhead Press | 6 each |
| 3 | Racked Squat | 8 each |
| 4 | Single-Arm Row | 10 each |
| 5 | Clean | 5 each |
| 6 | Farmer's Carry | 60s |

Rest 3 min. Repeat 3–4 rounds.

---

### Circuit 3 — The Storm

Ballistic and explosive. Hip drive powers everything.

| Order | Exercise | Reps |
|---|---|---|
| 1 | Two-Handed Swing | 20 |
| 2 | Clean & Push Press | 6 each |
| 3 | Snatch | 5 each |
| 4 | Alternating Swing | 16 |
| 5 | Thruster | 8 |
| 6 | Farmer's Carry | 45s |

Rest 2–3 min. Repeat 3–5 rounds.
